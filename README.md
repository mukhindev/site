## src/textTransformer

Преобразование текста в ReactElement-ы, с помощью цепочки трансформеров.

В трансформере описывается:

1. Как из текста извлечь фрагмент-сущность (например, заголовок, параграф, таблица и т.д.)
2. Как из текстового фрагмента извлечь полезные данные (например, для заголовка сам заголовок и уровень заголовка)
3. Какой ReactElement сформировать, используя полученные данные

Порядок трансформеров важен. Когда один трансформер находит фрагмент,
он его извлекает из источника и следующий трансформер его не увидит.

Рекомендуется располагать более сложные фрагменты в начале цепочки.

```TypeScript
// Цепочка фрагментов
const transformers = [
  markdownTableTransformer,
  markdownCodeTransformer,
  markdownParagraphTransformer,
];

// Ноды полученные прогоном через трансформеры (в каждой ноде храниться полученное состояние)
const nodes = parsePlainTextToNodes(markdown, transformers);
// Получение массива ReactElement-ов из нод
const elements = nodes.map((node) => node.render());
```
